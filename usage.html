ConsoleApplication1.exe Usage Guide

Step 1: Open the Command Line

  1. Press Win + R, type cmd, and press Enter to open Command Prompt.
  2. Navigate to the folder where ConsoleApplication1.exe is located. For example: 
    # Switch to D:\ConsoleApplication1 (adjust the path to your folder)
cd /d D:\ConsoleApplication1

Step 2: Use the Tool (Extract/Pack)

Option 1: Extract Files from a BND File

Command Format:

ConsoleApplication1.exe -e [PATH_TO_BND_FILE] [OUTPUT_FOLDER_NAME]

Example:

To extract test.bnd (in the same folder) into a new folder named output:

ConsoleApplication1.exe -e test.bnd output
  • The tool will automatically create the output folder and extract all files (named 00000, 00001, etc.) into it.

Option 2: Pack Files into a BND File

Prerequisite: Prepare a folder (e.g., input_files) containing files named in sequence (e.g., 00000, 00001, 00002...).

Command Format:

ConsoleApplication1.exe -p [INPUT_FOLDER_PATH] [OUTPUT_BND_FILE_NAME]

Example:

To pack files in input_files into a new BND file named new_test.bnd:

ConsoleApplication1.exe -p input_files new_test.bnd

Troubleshooting

  • "File not found" error: Ensure the file/folder path is correct (avoid spaces/chinese characters in paths).
